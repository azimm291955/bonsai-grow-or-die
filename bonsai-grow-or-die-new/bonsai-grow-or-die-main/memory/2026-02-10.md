# 2026-02-10

## Bonsai Yieldbook Planning Session

Aaron greenlit full planning and build of Bonsai Yieldbook — the real app, not just the Next.js placeholder.

### Vision
- **"Bonsai Yieldbook - A Place for Growers"**
- Data warehouse: manual yield input + Aroya (nutrient/feed) + Argus (temp/humidity) + METRC (plant lifecycle)
- Mobile-first PWA (no native app — cannabis apps get rejected from app stores)
- Read-only OpenClaw embedded for natural language data queries
- Multi-user with role-based access (Admin, Grower, Trimmer, Harvest Manager, Trim Manager)

### Key Decisions
- **No financials in the app** — purely yield and cultivation data
- **$45/mo hosting budget approved** (Supabase Pro $25 + Vercel Pro $20)
- **Branding:** Match bonsaicultivation.com colors and graphics. Make it eye-catching!
- **Historical data:** Load all 2023-2025 data from existing Google Sheets
- **BuddyPunch integration** needed for trimmer hours (TIP = lbs/day)
- **Incentive programs:**
  - TIP (Trimmer Incentive Program) — trimmers see their own lbs/day
  - Grow bonus — growers see room yields and bonus structure
- **Individual logins** — each person assigned to grow or trim team

### User Roles
- Admin (Aaron): sees everything, inputs everything + settings
- Grower: all yields live, room performance, bonus structure
- Trimmer: only their own TIP figures
- Harvest Manager: harvest data + room stats, inputs harvest sheet
- Trim Manager: all trim team data + productivity, inputs trimmer tracker

### Tech Stack
- Next.js + Tailwind (PWA)
- Supabase (PostgreSQL + Auth + Row-level security)
- Vercel hosting
- Recharts/D3 for visualizations
- OpenClaw embedded (Phase 4)

### Phases
- Phase 1: DB schema, auth, data input forms, historical upload, PWA (2 weeks)
- Phase 2: Analytics dashboard, production chart, room/strain views (2 weeks)
- Phase 3: METRC, Argus, Aroya integrations (timeline TBD on API access)
- Phase 4: Embedded OpenClaw AI chat

### Data Sources
- Manual: yield, trim, harvest (replaces Google Sheets input)
- Aroya: nutrient feed, substrate data (not in-house yet)
- Argus: temp, humidity, CO2 (Aaron checking on API)
- METRC: plant lifecycle, compliance (keys exist, untested)
- BuddyPunch: trimmer hours worked

### Integrations to Research
- BuddyPunch API for hours
- Argus API (Aaron checking)
- METRC API (keys on hand)
