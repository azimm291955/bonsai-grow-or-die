## The Math: Why Scaling Costs Matter

Each tier bought on an additional room costs 1.4x the previous:

```
Cost = Base Cost × 1.4^(n-1)

Where n = number of rooms with this tier
```

For Genetics T3 ($500K base):
- Room 1: $500K × 1.4⁰ = $500K
- Room 2: $500K × 1.4¹ = $700K (40% more!)
- Room 3: $500K × 1.4² = $980K (40% more!)
- Room 4: $500K × 1.4³ = $1,372K (40% more!)
```

The problem: You're adding **40% more cost** for each room, but each room generates the same **absolute revenue**. So diminishing returns kick in fast.

**Room 7 costs an extra $3.75M but only generates $0.8M/year extra = 4.6 year payback**
