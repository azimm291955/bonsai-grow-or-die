### What It Does
Horizontal Gantt-style timeline showing all room turns across the facility:
- Each room is a row; each turn is a colored bar on the timeline
- **Actual turns** (solid bars) — derived from Room_Production sheet data
  - Turn start estimated from previous trim completion + 7 day flip/clean time
  - End date = trim date (or harvest + 7 if no trim date recorded)
- **Projected turns** (striped/hatched bars) — extrapolated from room cycle patterns
- **Today marker** — red vertical line for temporal reference
- **Month gridlines** — background grid aligned to month boundaries
- **Interactive tooltips** — hover any bar for start/harvest/trim dates, yield (lbs), g/plant, plant count
- Color-coded by room matching existing palette
- Legend with actual vs projected vs today indicators

### Why It Matters
- Instantly visualize room utilization — see gaps (idle time) between turns
- Spot overlapping harvests to coordinate trim team scheduling
- Projected future turns help plan 3-4 months ahead
- CFO-friendly: one glance tells if facility is running at capacity
- Natural complement to the existing harvest calendar and cycle data tables
