### What It Does
Interactive heatmap matrix showing g/plant for every strain-room combination:

**Summary cards:**
- Avg g/plant across all combos
- Min–max range
- Best strain-room combination (highest g/plant)
- Most versatile strain (grown successfully in most rooms)

**Heatmap grid:**
- Rows = strains (top 20), columns = rooms
- Color-coded cells: red (low) → gold → green → emerald (high)
- Empty cells for strain-room combos with no data
- Row averages (per strain) and column averages (per room)

**Controls:**
- Metric toggle: g/plant vs plant count
- Sort: by best g/plant, most production, or alphabetical
- Min-plant filter: 1/3/5/10/20 — filter out low-sample combos
- Hover: detail bar showing exact values + comparison vs strain avg

**Color legend** at bottom explains the heat scale

### Why It Matters
- Answers "which strains should go in which rooms?" — critical planting decision
- Combines two dimensions (strain performance + room performance) into one visual
- Min-plant filter prevents small-sample noise from skewing decisions
- CFO can see at a glance where to optimize strain placement for maximum yield
- Uses existing `strainRoomPerformance` data — no new API calls needed
